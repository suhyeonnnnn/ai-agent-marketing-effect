Note on product_id vs product_number:
  - text_json: "product_number" (position 1-8), NOT product_id
  - text_flat: "[Product N]" header = position number
  - html: data-product-id attribute = actual product_id (but hidden in DOM)
  - screenshot: no ID visible at all

  → LLM은 "chosen_product_number" (position)로 응답.
    API route에서 position → actual product_id로 매핑.
